# PULA Biz direct package downloads

Upload signed native packages here with these names so the website buttons work:

- `pula-biz-latest.apk`
- `pula-biz-mac-latest.dmg`

Optional future packages:

- `pula-biz-windows-latest.exe`
- `pula-biz-windows-latest.msi`

`pula-biz-mac-latest.dmg` is currently generated from a lightweight Mac WebView wrapper.
`pula-biz-latest.apk` still needs an Android build toolchain before it can be generated.
